Feedback gathering reconfiguration Enactor
------------------------------------------

1- Obtain this project by cloning Monitoring and Feedback reconfiguration
git clone https://github.com/supersede-project/monitor_feedback_reconfig.git
This creates a directory into your filesystem, hereafter referenced as FEEDBACK_NACTOR_HOME
2- Install Gradle in your system
3- Go to $FGENACTOR_HOME/IF/API/eu.supersede.feedbackgathering.reconfiguration.enactor
4- Execute: gradle eclipse
5- Launch Eclipse, import eu.supersede.if.api.test project as Gradle project: File/Import/Gradle/Gradle project
6- Execute Test cases in src/test/java/eu.supersede.feedbackgathering.reconfiguration.enactor.FeedbackEnactor.FeedbackEnactorTest.
We tested the application = 4, you can change this ID.
